  • 설명

    클라이언트, 서버, DB까지 3-tier를 풀스택으로 개발한 모바일앱 입니다.
    로그인, 회원가입, 로그아웃 페이지를 맡아서 개발했습니다.

    주요 기능

    회원가입 페이지에서 각 필수사항을 입력해야 회원가입이 되도록 하였습니다.
    중복체크 부분은 DB와 통신하여 중복가입을 방지하도록 하였습니다.
    비밀번호 입력은 비밀번호 확인칸과 일치하지 않으면 테두리가 빨간색이 되도록 하였습니다.
    전화번호 입력칸은 정규표현식을 이용해서 자동으로 '-' 이 생기도록 하였습니다.
    개인정보 처리방침 동의약관은 모달창으로 만들었습니다.
    회원가입 성공시 비밀번호를 해쉬로 저장하도록 하였습니다.
    로그인 페이지에서 DB에 회원정보가 없으면 로그인을 하지 못하게 막았고,
    로그인 성공시 asyncStorage에 로그인 정보를 저장시켰습니다.
    로그아웃은 asyncStorage에 로그인 정보를 삭제시키고
    로그인 페이지로 돌아가도록 하였습니다.
    DB는 pool connection으로 연결하여 DB의 안정성을 높였습니다.

    사용 스택

    React Native, React Navigation, Node.js, Express, MySQL, Expo
